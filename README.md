# SteamSharing
Эта программа позволяет играть с другом в одну игру одновременно через Steam, путём перезапуска вашей игры каждые ~5 минут. Перед запуском укажите путь к игре и процесс. Ваш друг должен первым запустить игру и настроить сетевой сервер, а вы после перезапуска через комбинацию клавиш Shift+Tab можете присоедениться к другу.
